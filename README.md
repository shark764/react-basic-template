## Basic template

_Clone this project and rename it._

```sh
git clone git@github.com:shark764/react-basic-template.git <project-name>
```

#### Rename project in package.json
```json
{
  "name": "project-name",
  "version": "0.1.0",
  ...
}
```

#### Install dependencies
```sh
npm install
```

#### Format with prettier
```sh
npm run format
```

#### Format with eslint
```sh
npm run lint:fix
```
